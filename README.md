# Alterations in chromatin accessibility and conformation elucidate genetic mechanisms in ASD

**Jiani Yin, Jerry Huang, Jing Ou, George Chen, Lucy K. Bicks, Brie Wamsley, Yuyan Cheng, Lawrence Chen, Jillian R. Haney, Rain Wong, Daniel H. Geschwind**

Center for Autism Research and Treatment / Program in Neurogenetics, David Geffen School of Medicine, UCLA

📄 Preprint: [medRxiv 2025.04.02.25325129](https://www.medrxiv.org/content/10.1101/2025.04.02.25325129v1.full)

---

## Overview

This repository contains the R and shell scripts used to generate all main and supplementary figures in the manuscript. We profiled **chromatin accessibility (ATAC-seq)** and **3D genome architecture (Hi-C)** in post-mortem cerebral cortex from 53 ASD and neurotypical individuals, including 7 cases with Dup15q syndrome.

Key findings:
- Genome-wide **differentially accessible regions (DARs)** in ASD brain, validated against H3K27ac data
- Reduced **CTCF binding** drives decreased chromatin accessibility at distal enhancers and weakened **TAD boundaries**
- Up-regulated **AP-1/NFE2 transcription factors** increase accessibility at promoter regions in astrocytes
- **caQTLs** and **rare de novo SNVs** within ASD down-regulated distal ATAC peaks increase disease risk
- **DNA hypermethylation** at weakened TAD boundaries (enriched for CTCF binding sites), linked to mQTLs increasing ASD risk
- CRISPR validation of regulatory relationships at three ASD risk loci (TRMT1/NACC1/NFIX, ADSL/TNRC6B, PIAS4)

## Repository Structure

```
.
├── data/                                 # Reference files and metadata
├── plots/                                # Output figures
├── results/                              # Processed results files
├── scripts/                              # All analysis scripts, organized by figure
│   ├── 01_chromatin_accessibility/       # ATAC-seq analysis, DAR identification, CHART method
│   ├── 02_transcription_factors/         # TF motif enrichment (HOMER), footprinting (TOBIAS)
│   ├── 03_TF_networks_target_genes/      # TF networks, PPI (DAPPLE/STRING), GO & cell type enrichment
│   ├── 04_genetic_variants/              # caQTLs, rare SNVs, CRISPR validation, genome browser tracks
│   ├── 05_TAD_boundaries_methylation/    # Hi-C TAD analysis, DNA methylation, mQTLs
│   ├── 06_working_model/                 # Integration of epigenetic layers, DEG contributions
│   └── 07_review_analyses/               # Additional analyses added during peer review
└── utils/                                # Utility functions and helper scripts
```

## Data Dependencies

Scripts in this repository load `.rda` and other intermediate data files generated by upstream bioinformatic pipelines (ATAC-seq processing, Hi-C processing via HiC-Pro, FitHiC2, GENOVA, FIREcaller). These data files are not included in this repository.

| Analysis | Tool | Version |

## Data Availability

Raw sequencing data has been deposited in the NIMH Data Archive (NDA) under the 'PsychENCODE (PEC) Integrative Genomic Analysis of Human Brain Development and Autism' collection (Collection 2467). Supplementary tables containing peak coordinates, chromatin loop calls, DAR results, and CRISPR primer sequences are available in the manuscript.
